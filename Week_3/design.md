# Create a Character Walking Animation

### Description
Bring a character to life by creating a walking animation. Your animation should show smooth movement, good timing, and a sense of personality. Focus on making the walk cycle look natural and engaging.

### Things to be Included
- A single character
- A seamless looping walk cycle
- Smooth and realistic movement
- Clear personality or style in the character's walk

### Rules
- Create a looping walk cycle animation for one character
- The animation must be 3 to 5 seconds long
- Use either 12 FPS or 24 FPS
- Any animation style is allowed (2D, 3D, pixel art, frame-by-frame)
- The animation must be your own work
- You may use pre-made character rigs, but all animation and keyframes must be created by you
- Submit the final animation as an MP4 video
- Submit the project file along with the final video
- Submit screenshots showing your animation workflow

### Points
- This task carries 10 points

# Design a Soap Box Package

### Description
Design the outer packaging for a soap product. Create a unique soap brand and design a package that is attractive, creative, and suitable for your chosen theme. Think about how real soap products are presented in stores.

### Things to be Included
- A fictional brand name and logo
- A product name and scent (e.g., Lavender Mist, Charcoal Detox)
- A chosen theme for the soap brand (e.g., natural, luxury, sporty, or kids)
- A net weight placeholder (e.g., Net Wt. 100g)
- Space for ingredients, directions, and a barcode on the back
- A complete flat packaging layout showing the front, back, and side panels

### Rules
- Create a 2D flat template of the soap box package
- You may use Photoshop, Illustrator, Canva, or similar design software
- All artwork, branding, and layouts must be your own original work
- The package design should match the theme you have chosen
- Submit the final design as a PNG, JPG, or PDF file
- Submit the project file along with the final design

### Points
- This task carries 10 points
