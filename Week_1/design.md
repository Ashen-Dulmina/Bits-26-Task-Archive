# Design a Sci-Fi Movie Poster

### Description

Create an attractive poster for a sci-fi movie. Your poster should grab people's attention and make them interested in watching the movie.

### Things to be Included

- A big movie title that is easy to notice
- A short and catchy tagline about the movie
- A main image or artwork that shows the sci-fi theme
- Suitable colors, fonts, and design elements to make the poster look attractive

### Rules

- Poster size must be 1080 × 1350 pixels (4:5 ratio)
- You may use any design tool such as Canva, Photoshop, Illustrator, or others
- The design must be your own work and should not copy an existing movie poster
- Submit the final design as a PNG, JPG, or PDF file

### Points

- This task carries 10 points

### Things to be Included

# Create a Low-Poly Tree Model

### Description

Create a 3D model of a tree using a low-poly style. Use simple shapes and visible polygons instead of realistic details.

### Things to be Included

- A tree model created in a low-poly style
- A trunk and leaves made using simple polygon shapes
- A clean and balanced overall design
- Solid colors or simple color gradients

### Rules

- Keep the model under 1,000 polygons (faces)
- Use Blender to create the model
- Do not use high-resolution or realistic textures
- Submit a rendered image of the model in PNG or JPG format
- Submit the original Blender (.blend) file along with the rendered image

### Points

- This task carries 10 points
