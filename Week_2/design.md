# Create a Splash Screen Animation

### Description
Create a splash screen animation for a mobile application. The animation should be short, attractive, and suitable for an app's opening screen. Use creative motion and visual effects to make the animation engaging.

### Things to be Included
- An app name or logo
- Smooth animation effects
- A clean and attractive design
- A background that matches the app theme

### Rules
- Video size must be in 9:16 portrait format
- You may use any video editing or animation software such as Adobe Premiere Pro, After Effects, CapCut, or similar tools
- The animation must be your own original work
- Do not copy layouts, animations, or styles directly from YouTube or other sources
- Create your own design, layout, and animation style

### Points
- This task carries 10 points
